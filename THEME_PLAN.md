> Тема объявлена. Этот документ — отбор из всего набросанного на брейнсторме.
> Что берём, что режем, где риски, кто что делает в ближайшие часы.
> Параллельно ребята делают **схему всех действий в мире** (action map по списку глаголов) — этот документ опирается на неё, не дублирует.

---

## 1. Концепт после темы

ГГ — девушка из обычного мира, перепрыгивает в фэнтези-миры (с просачиванием реальной жизни). На каждой арене — группа врагов с уникальным набором абилок. Убиваешь — **поглощаешь** один аспект (абилку или пассивку) на выбор. С каждым поглощением сдвигается **моральный компас**, и появляются голоса в голове — диалоги поглощённых сущностей. Сюжетный слой ведёт к точке «мы что, злодеи?».

Идентичность концепта (плёнка / эскалация голоса / мета-ирония) **остаётся** — ложится на тему один в один.

## 2. Ядро — без этого нет игры

Семь штук. Если одна не работает к утру субботы — игры нет.

| # | Механика | Кто несёт |
|---|---|---|
| 1 | **Hex-арена** + движение ПКМ + мир тикает по ходу игрока | Егор |
| 2 | **Враг = набор абилок.** 1–3 абилки на врага, общий registry | Егор + Сергей |
| 3 | **Ability registry.** Абилка = JSON + GDScript. Один и тот же контракт для игрока и врага. | Сергей |
| 4 | **Поглощение.** После боя — выбор 1 из N доступных абилок убитых. UI выбора. | Сергей + Алексей |
| 5 | **Слот-инвентарь героини.** Стартует с 1–2, растёт по run'у. Слотов лимит. | Сергей |
| 6 | **Моральный компас.** ОДНА ось 0–100, 3–4 пороговых ивента. autoload. | Алексей |
| 7 | **Roguelike loop.** Арена → выбор поглощения → портал → следующая арена → босс. | Алексей |

Поверх — **DialogueManager** (Алексей, engine) + **контент диалогов и голосов в голове** (Никита).

## 4. Антискоуп — что точно не делаем

Из брейнсторма набралось много, режу беспощадно.

**Из списка глаголов** — режем как мусор для джема: танцевать, спать, делать бочку, рожать, прорастать, стареть/молодеть, отматывать время, ставить мины, перекодировать ткань бытия, неевклидовы головоломки, ультимативные «суперабилки», давать плоды.

**Из списка локаций** — берём максимум **3 мира** на джем. Кандидаты по принципу «дёшево для художки и контрастно»: офис (скучный реал) + ад (фэнтези) + один странный (клавиатура / нарисованный примитив). Остальные миры — **never** на джем, в метапрогрессии не разблокировать.

**Из списка врагов** — 8–10 типов на джем, не больше. Каждый поставляет 2–3 абилки в registry. Не делаем кроненбергов, тихоходок, инопланетян, единорогов, смешариков на старте.

**Из идей механики:**
- Континуальный моральный компас с эффектами на каждый шаг → ОДНА ось, 3 порога.
- Постоянно меняющиеся арены → только арена босса.
- Многомерные dialogue trees с состоянием → 1 уровень выбора (правило из CLAUDE.md).
- «Защитник деревни → пожирай сограждан» — отдельная сюжетная ветка → один сценарий, не вариативность.
- Реал-time переход между мирами в одном run'е → между run'ами / актами, не внутри.

## 5. Риски — где может рвануть

### P1 — блокеры, решаем сегодня вечером

**R1. Абилки в рантайме.**
Поглощённая абилка должна работать у игрока **так же**, как у врага. Если каждая абилка — это behavior tree или специальный класс, не успеем.
**Митигация:** все абилки реализованы по единому контракту `Ability` (GDScript), принимают `caster: Node`, не различают player/enemy. Параметры — в JSON. Никаких особых классов под игрока. Сергей пишет интерфейс **до** того, как кто-то начнёт делать конкретные абилки.

**R2. Множество мировоззрений × множество абилок.**
«Мобы реагируют на твой моральный компас» — комбинаторный взрыв, если делать в лоб.
**Митигация:** компас — 1 ось, 3 порога (например, human / corrupted / lost). Реакции — на пороги, не на значения. Максимум 3 ветки реакций на врага.

**R3. Поглощение требует UI выбора + понятного описания «что я получу».**
Контент-heavy и легко непонятно для игрока.
**Митигация:** карточки в стиле Slay the Spire (иконка + имя + 1 строка эффекта). Стасян пишет описания вместе с registry.

### P2 — управляемо, но следим

**R4. Арт боттлнек.**
3 мира × 8 врагов × объекты = много ассетов. Катя одна.
**Митигация:** один из миров — намеренно «нарисованный примитив» (бумажный) — почти ноль работы для художки, плюс это укладывается в тему («перепрыгивает в нарисованный мир»). Враги переиспользуют силуэты между мирами с реколором.

**R5. Голоса в голове vs внешние диалоги — две системы тона.**
**Митигация:** ОДИН DialogueManager, голоса в голове — это специальный `speaker` с другим визуалом портрета (без портрета / силуэт). Никита пишет с разной типографикой, но один и тот же engine.

**R6. «Босс меняет арену».**
Звучит круто, реализация — runtime hex regen, отдельный код. Риск, что застрянет на сутки.
**Митигация:** один босс, простая версия (некоторые гексы становятся непроходимыми по триггеру). Если за 4 часа не получается — режем до «босс ставит шипы на гексы», без structural изменений.

### P3 — мониторим

**R7. Финальный приём** (скип off, реальный голос) — записать реальный голос требует Андрея перед микрофоном в субботу. Запасной вариант — один из ребят пишет на телефон.

**R8. Интерактивное окружение** манит расширяться. Договорённость: **не более 7 типов объектов** до утра субботы. Расширение — в воскресенье уже не будет.

## 7. Спорные точки — нужны решения сегодня

Голосуем сейчас, чтобы не пересобирать утром.

1. **Стартовый набор героини.** 0 абилок (всё собирается с боев) / 1 базовая (например, удар) / 4 как было (ослабленные)? Влияет на первые 5 минут игры — самые важные.
   *Моя ставка: 1 базовая (move + slot 1 = удар), остальное собирается.*

2. **Что значит «поглощение пассивки» технически.** Это абилка в slot? Отдельный список модификаторов на героиню? Я бы свёл всё к одному: всё, что собираешь — это **пункт в общем списке**, активные идут в QWER-слоты, пассивные просто висят. Один контракт — проще на джеме.

3. **Сколько слотов под активные абилки.** 4 (QWER) фикс? Растёт по run'у? Расширяется в метапрогрессии?
   *Моя ставка: 4 фикс на джем. Потом думаем.*

4. **Моральный компас — какие именно 3 порога и какие эффекты на каждом.** Это нужно зафиксировать ДО того, как Никита пишет реплики. Ставка: human → corrupted → lost, эффекты — изменения портрета ГГ + новые опции в диалогах + появление голосов в голове.

5. **Action map от ребят** — когда готов? Из него мы уточним registry абилок. Если позже сегодня — Сергей делает первый proof-of-concept на 5 заглушках, потом расширяет.
