# egor/ — personal scratch space

Personal notes, drafts, prompts, WIP, anything that helps **egor** work but doesn't belong in shared code or specs yet.

Each Claude session reads this folder at startup (per `PROJECT_INSTRUCTIONS.md`). Put context here that you want Claude to pick up automatically — e.g. "current task focus", "pending decisions", "things I keep forgetting".

Anything that becomes shared (specs, designs, content) should be promoted to the appropriate top-level folder (`specs/`, `data/`, `scripts/`, etc.) via a PR.
