# GameJamProject — План спеков и задач

Состояние на 5 мая 2026. Реконструировано из чата 4 мая + дополнено результатами сессии по дизайн-докам.

---

## Легенда типов

- **[spec-S/M/L]** — отдельный спек по полному циклу (Clarify → Spec → Plan → Tasks → Implement → Polish). Размер: S — 1 день, M — 3-5 дней, L — больше недели.
- **[task-S/M/L]** — без полного спека, обычная задача. S — пара часов, L — пара дней.
- **[chore]** — инфраструктурная мелочёвка вне фич.
- **[research]** — нужно подумать прежде чем планировать.
- **[process]** — не разовая задача, а повторяемая работа.
- **[defer]** — отложено сознательно.
- **[DONE YYYY-MM-DD]** — выполнено, оставлено для истории.
