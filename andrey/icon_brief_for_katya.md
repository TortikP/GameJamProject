# Бриф для Кати — иконки скиллов и статусов

## Размер и формат

**128×128 PNG, прозрачный фон, квадрат.**

- Основное изображение в круге ~110×110, по краям 8–10 px прозрачного запаса (чтобы не резалось при кропе/рамке).
- Один мастер-файл на иконку — движок сам уменьшит:
  - `64×64` — карточки выбора скилла после волны (`SkillOfferCard`)
  - `20×20` — пилюли статусов над персонажами (`StatusIconStrip`)
- Из-за 20-px пиллов **статус-иконки лучше делать читаемыми силуэтом**: одна крупная форма + 1 акцентный цвет. Детализация теряется при ужатии в 6 раз.
- Скилл-иконки могут быть детальнее (карточка 64-px терпит больше).
- Пишем в `assets/icons/skills/<id>.png` и `assets/icons/statuses/<id>.png`.

## Статус-эффекты (12 шт.)

| id | Название | Что делает (для иконки) |
|---|---|---|
| `burning` | Горение | Цель горит, тикает урон огнём каждый ход |
| `poisoned` | Яд | Цель отравлена, тикает % урона каждый ход |
| `shielded` | Щит | Поглощает N следующих ударов |
| `stunned` | Оглушение | Пропускает ход |
| `slowed` | Замедление | Двигается медленнее |
| `rooted` | Обездвижен | Не может двигаться, но может атаковать |
| `feared` | Страх | Бежит от источника, теряет контроль |
| `enraged` | Ярость | Бьёт сильнее, агрессивнее |
| `strong` | Сила | Наносит больше урона |
| `weak` | Слабость | Наносит меньше урона |
| `glitched` | Сбой | Ведёт себя непредсказуемо, артефакт/глитч |
| `summoned` | Призван | Маркер призванного существа (опционально, можно не рисовать) |

## Технические детали (на всякий случай)

- Кода-плейсхолдеров для иконок сейчас нет — пока файла в `assets/icons/<...>` не будет, рендерится unicode-глиф или пустой `TextureRect`. Иконку можно подкладывать поштучно по мере готовности, всё остальное продолжит работать.
- Стиль — на твоё усмотрение, но желательно тот же, что у `hex_atlas` (painterly с тёплым контуром), чтобы UI не дрался с ареной.
- Если есть сомнения по конкретной иконке — скинь набросок, утвердим и не будем переделывать.
