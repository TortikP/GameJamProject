**Status:** Reserved — содержание заполняется утром, вне этой сессии.

## Цель (одно предложение)

Допилить map editor (020) по UX и закрыть оставшиеся баги, которые всплывут на плейтесте у команды.

## Что вероятно войдёт (черновой список — фильтровать утром после плейтеста)

- Drag-and-drop существующих объектов (T019 из 020 — был помечен как P3 stretch, отрезан).
- Box-select / multi-select для batch-операций (был в out_of_scope, может вернуться).
- Undo/redo (был в out_of_scope; если уложимся в простой stack-based — стоит включить).
- Канва не фиксированная 25×25 — кнопка «расширить на N» (тоже был out_of_scope).
- Connectivity warning — подсветка изолированных островов перед save (visual cue, не блокер).
- Spawner: per-actor overrides (HP/skills для конкретного манекена в карте).
- TileObject: визуальная подсветка blocks_movement (полупрозрачная сетка) и has_effect (пульсация цветом эффекта).
- Hotkey list (G — toggle grid, +/− zoom, etc.) с overlay'ем-подсказкой.
- Preview-тumbnail карт в Load FileDialog.
- Better placeholder art для объектов (сейчас цветные квадраты по hash'у — норм, но цвета бывают неудобные).
- Возможно: вторая `LevelData` версия схемы для backward-compat миграции.

## Зависимости

- 022 (hex-shape) — желательно стабилизировать форму до полировки UI вокруг гексов.
- Список багов — наполняется по мере плейтеста команды.
