# 004-godmode-base — smoke test

Прогнать перед мерджем PR. Всё в Godot 4.6.2, F5 запуск проекта.

## 0. Boot

- [ ] Проект запускается без ошибок в логах.
- [ ] В консоли видно `[INFO][AbilityDatabase] loaded 1 abilities` (debug_punch).
- [ ] В `main.tscn` видна кнопка "▶  Godmode (sandbox)" под "Arena Demo".

## 1. Arena demo не сломан

- [ ] Кнопка "▶  Arena Demo" → открывается старая демка Егора. Q/W/E/A/S/D ходят. ЛКМ-путь работает.
- [ ] Возврат в main по любому способу (ESC или перезапуск).

## 2. Godmode boot

- [ ] Кнопка "▶  Godmode (sandbox)" → открывается `scenes/dev/godmode.tscn`.
- [ ] Видна гекс-сетка 10×10 (вся трава, без стен).
- [ ] Игрок (синий гекс) в центре сетки.
- [ ] Сверху-слева: `Turn: 1`. Подсказка: `RMB=move  LMB/QWER/1234=cast  F1=spawn dummy  F2=clear`.
- [ ] Внизу-центр: 4 кнопки `Q W E R`. В Q написано `debug_punch`. Q подсвечена (она активная).
- [ ] В консоли: `[INFO][Godmode] ready. ...` + `[INFO][Godmode] Player at (5, 5)`.

## 3. Движение игрока (RMB)

- [ ] RMB по соседнему проходимому гексу → игрок плавно идёт на него.
- [ ] После завершения шага: `Turn: 2` в HUD.
- [ ] RMB по дальнему гексу → игрок проходит весь путь, на каждый шаг анимация. После окончания пути `Turn` инкрементируется **на 1** (не на длину пути).
- [ ] RMB по своему гексу → ничего не происходит, тик не идёт.
- [ ] RMB по гексу за пределами карты → ничего не происходит, тик не идёт.

## 4. Спавн манекенов (F1)

- [ ] Навёл курсор на свободный гекс, F1 → красный гекс-манекен появляется. Лог `[INFO][Godmode] spawned dummy_001 at ...`.
- [ ] `Turn` **не** инкрементируется (спавн dev-инструмент, не действие игрока).
- [ ] F1 на занятый/непроходимый гекс → лог `[INFO][Godmode] cannot spawn at ...` или `occupied at ...`. Манекен не появляется.
- [ ] Можно наспавнить 5+ манекенов.

## 5. Каст ability

- [ ] Q активна по умолчанию. Курсор на манекена, ЛКМ → `[INFO][Actor] dummy_001 -5 hp (15/20)`. `Turn` +1.
- [ ] Альтернатива: курсор на манекена, нажать Q (или 1) → тот же эффект.
- [ ] ЛКМ по пустому гексу → лог `[INFO][Ability] debug_punch: no targets` (или `[INFO][Godmode] no valid target`). `Turn` **не** инкрементируется.
- [ ] ЛКМ по самому игроку (нажать Q стоя на месте) → лог "no targets". Тик не идёт.
- [ ] 4 каста подряд по тому же манекену → на 4-м касте: `[INFO][Actor] dummy_001 -5 hp (0/20)` + `[INFO][Actor] dummy_001 died`. Манекен исчезает, освобождает гекс.

## 6. Очистка арены (F2)

- [ ] Наспавнить 3+ манекенов, F2 → все красные гексы исчезают одновременно. Лог `[INFO][Godmode] cleared 3 manekins`.
- [ ] `Turn` **не** инкрементируется.
- [ ] После очистки можно снова спавнить и кастовать.

## 7. Hot-reload game_speed.cfg

- [ ] Запустить godmode, поменять в `config/game_speed.cfg` `[arena] step_duration=0.5`, нажать `F6` (reload_speed_config).
- [ ] RMB-move заметно медленнее.

## Что должно быть в логах суммарно (пример сессии)

```
[INFO][TurnManager] reset → 1
[INFO][AbilityDatabase] loaded 1 abilities
[INFO][Godmode] Player at (5, 5)
[INFO][Godmode] ready. ...
[INFO][Godmode] spawned dummy_001 at (6, 5)
[INFO][Actor] dummy_001 -5 hp (15/20)
[INFO][Actor] dummy_001 -5 hp (10/20)
[INFO][Actor] dummy_001 -5 hp (5/20)
[INFO][Actor] dummy_001 -5 hp (0/20)
[INFO][Actor] dummy_001 died
```

## Известные ограничения (НЕ баги в скоупе этого PR)

- 1/2/3/4 — алиасы Q/W/E/R, не отдельные слоты loadout. Будет в следующем PR.
- Манекены не двигаются и не отвечают (нет AI). Тоже следующий PR.
- Нет анимации каста (мгновенный урон). Polish-фича позже.
- Нет визуализации HP. Манекен умирает молча — только в логах.
- Если кликать RMB во время уже идущего движения, второй клик игнорируется (`grid._moving` lock). Очередь — отдельная задача.
