# 030-enemy-ai-smart-targeting — SMOKE tests

Ручная проверка в Godot. Один тест = одна строка результата.
Формат: `[PASS/FAIL/SKIP] #N — описание. (timestamp)`

Запускать на ветке `alexey/030-smart-targeting` после merge в staging.

---

| # | Что проверяем | Ожидаемый результат |
|---|---|---|
| 1 | backward-compat: default_melee, enraged, feared | Поведение мании/страха без изменений |
| 2 | 3 melee_fighter идут к игроку | Занимают разные клетки, не стакаются |
| 3 | ranged_mage с AoE-скиллом, игрок близко | Rule 2: бьёт в незаконтестованный соседний hex |
| 4 | ranged_mage без hex-скилла (только ActorTarget damage) | Rule 2 не срабатывает, Rule 3: атакует напрямую |
| 5 | healer <40% HP, есть раненый союзник | Лечит СЕБЯ, не союзника |
| 6 | healer >40% HP, союзник <60% HP в радиусе 3 | Лечит союзника, не атакует |
| 7 | buffer с союзниками разного HP | Баффает союзника с наибольшим HP |
| 8 | selector_target_without_status: один враг уже дебафан | Дебафает другого врага |
| 9 | ranged_mage, все соседние hex игрока заняты | Rule 3: атакует напрямую |
