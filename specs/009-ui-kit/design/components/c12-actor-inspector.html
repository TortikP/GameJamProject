Spec — C12 Actor inspector panel
Purpose
Show the full data for whatever the player selected. Single component, two data-driven variants (actor / hex) and two interaction modes (play / god). Pillar 2: identical structure for player and enemies.
Anchor & sizing
Top-right, anchored to top + right edges. Width 320px fixed. Height grows with content; min height ~200, max height = viewport_h − top HUD − combat log. Internal sections scroll if overflow (one shared ScrollContainer).
Inputs
selection: { kind: "player" | "actor" | "hex" | "none",
             actor_id?: int,
             hex_coord?: Vector2i }
cast_mode: { state: "idle" | "casting", slot?: int }   # cast pin override
edit_mode: bool   # true in Godmode scene
data:      Actor | Hex | null  # resolved upstream from selection
Outputs
signal stat_changed(field: String, value: Variant)   # godmode only
signal occupant_link_clicked(actor_id: int)         # hex variant
signal skill_pip_hovered(skill_id: String, in: bool) # drives C14 tooltip
Edge cases
• selection.none: show empty placeholder, defaults back to player next frame.
• cast_mode.casting overrides selection — show skill info; restore selection panel on cast resolve/cancel.
• HP overflow / shield: show as 120/100 in numeric, bar caps at 100%.
• No skills bound: render empty pip slots with dashed border.
• Hex with multiple tile effects: wrap chips, no truncation.
• Focus trap (§7): Godmode SpinBoxes capture Q/W/E/R; controller MUST release focus on game-key keypress (get_viewport().gui_release_focus() on input). Yellow outline indicates field has focus.
Godot mapping
ActorInspector : PanelContainer       # anchors_preset=TOP_RIGHT
                                            # custom_minimum_size = (320, 200)
  VBoxContainer                          # separation = 0; sections add own padding
    HeaderRow: HBoxContainer            # name + team badge
      VBox {Label name, Label sub}
      Label team_badge                    # StyleBoxFlat colored border
    SectionHP: VBoxContainer
      Label "HP"                           # 10pt mono caps
      HBoxContainer
        Label hp_value                     # 24pt mono
        Label hp_meta                      # 11pt mono dim
      HpBar (custom _draw, 8px tall)        # reuses C4 with size override
    SectionStats: VBoxContainer
      StatRow ×N : HBoxContainer
        Label field
        Label value | SpinBox editor   # swap based on edit_mode
    SectionStatuses: HFlowContainer       # wraps chips
      StatusChip ×N
    SectionSkills: GridContainer (cols=4)
      SkillPip ×4                          # Button, drives C14 tooltip on hover
    SectionIntent (only if team==enemy)
      PanelContainer with sem-damage border
        RichTextLabel intent_desc           # bbcode_enabled
Mode toggle: edit_mode setter rebuilds StatRow children (Label↔SpinBox). Cast-mode override is a separate SkillInspector sibling that visible-toggles atop the actor view.
