C14 — Skill tooltip (rich)

Floating panel anchored to a skill pip / slot on hover (400ms delay). Shows ordered ability list with target/area, effects with modifier-adjusted numbers (10 → 15), and tags. Recommended numeric format: base → final.

States

Single ability · ready

Simple skill: one ability, one effect, no modifiers. Cooldown=0 → "ready" (green).

Slash READY · cd 1
1. target: entity · area: single
damage 10
melee physical

Modifier-adjusted · base→final

Sharpened modifier adds +5 dmg. Format: 10 → 15. Delta in blue.

Bloody Edge READY · cd 2
1. target: entity · area: single · range 3
damage 10 → 15+5
bleed · 2t 3/turn
melee physical + sharpened

Multi-ability composition

Skill = ordered abilities sharing one cooldown. Each ability shown separately, numbered.

Push Wave cd 3
1. target: direction · area: cone(3)
damage 8
push · 2 hex →
2. target: self · area: self
shielded · 2t +15 hp
aoe force

Cooling · 2 turns

Header shows turns remaining; ability list still readable.

Heal Pulse COOLING · 2 turns
1. target: hex · area: circle(2)
heal 8
regen · 3t 2/turn
heal aoe

Stacked modifiers

When multiple modifiers stack, math line at bottom: final = (base+adds) × muls.

Bloody Edge READY · cd 2
1. target: entity · single
damage 10 → 22×1.5 after +5
+ sharpened + rage

Spec — C14 Skill tooltip
Purpose
Show full skill definition on hover: ordered abilities, target/area, effects with base→final math, tags. The "depth on demand" layer (§10).
Anchor & sizing
Floating, anchored to the hovered slot/pip with 12px offset. Width 320 fixed; height grows with content. Auto-flips above/below to stay on screen.
Inputs
skill: Skill {
  id, name, cooldown_max, cooldown_remaining,
  abilities: Array[Ability],
  tags: Array[String]
}
modifier_breakdown: Array[{ effect_id, base, final, formula }]
Numeric format (recommended)
base → final with delta annotation. For multiplied stacks: final ×mul after +add in delta slot. Avoids arithmetic noise.
Edge cases
• Empty slot: minimal hint variant.
• Cooling: header turns orange + "COOLING · Nt", ability list still rendered.
• Long ability list (>3): tooltip grows — no scroll. Player only sees this on demand.
• Off-screen anchor: flip vertical / horizontal; never clip out of viewport.
