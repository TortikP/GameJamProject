Spec — C21 Run summary / death
Purpose
Close out a run. Tells the player what just happened (verdict + killer), how it went (stats), and what they keep (modifiers, voices). The bridge between this run and the next.
Anchor & sizing
Centered modal, max-width 960. Pause-the-world (modal protocol). Renders on top of everything except the title screen.
Inputs
verdict:        Enum { DEATH, VICTORY }
wave_index, wave_max
wave_label
killer:         { name, race }     # null on victory
stats:          { turns, dmg_dealt, dmg_taken, kills }
modifiers:      Array[Modifier]    # ordered by acquisition
voices:         Array[Voice]
seed:           String
Outputs
signal restart, signal title_screen
Edge cases
• Voices > 8: chips wrap; allow vertical scroll within the card; never paginate (run length is part of the read).
• 0 modifiers / 0 voices: show empty-state line ("You walked alone." / "No voices yet.") instead of an empty card.
• Self-kill / environmental death: killer line uses "the cellar took you"-style copy; no race tag.
