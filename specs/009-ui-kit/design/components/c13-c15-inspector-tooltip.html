UI KIT // C13 + C15

C13 Hex inspector · C15 Generic tooltip

C15 is the cheap one-line tooltip every generic UI element gets (button, icon, stat label) — title + 1-line meta + optional keybind. C13 is its richer cousin: the inspector subpanel that opens when you hover/click a hex with no entity (terrain, tile-FX, cover). Both share fonts and border treatment so a player builds one reading habit.

Spec — C15 Generic tooltip
Purpose
Default hover tooltip for any UI element that doesn't warrant C13/C14. Cheap and predictable.
Inputs
title:    String?    # null → caller passes raw text only
text:     String
meta:     String?    # mono uppercase line
keybind:  String?    # rendered as kbd chip after title
Edge cases
• Long text: wraps at 280px max-width; no internal scroll.
• Reduce-motion: appear/disappear instant instead of 80 ms fade.
• Anchor flip: auto-flip vertical/horizontal to stay on-screen.
Godot mapping
GenericTooltip : PanelContainer       # Control._make_custom_tooltip()
  VBoxContainer
    HBoxContainer head: Label title + Label kbd
    Label meta
    Label body
Spec — C13 Hex inspector
Purpose
The C12 sibling for empty hexes. When inspect-mode points at a hex with no entity, this opens in the same screen slot.
Anchor & sizing
Same anchor as C12 (right-side inspector slot). 280×auto. Replaces C12 when current selection has no entity.
Inputs
hex:      Hex
terrain:  Terrain { name, kind, move_cost, cover, elevation, passable }
effects:  Array[TileEffect]   # mirror of C5 vocabulary
Outputs
none — read-only.
Edge cases
• Impassable: hide the stats grid; show only the warning meta line.
• 0 effects: drop the effects block entirely (don't render an empty heading).
• Hex outside fog of war: show "?" coord, terrain "unseen", no stats. Inspector still opens so the affordance feels reliable.
Godot mapping
HexInspector : PanelContainer        # swaps with C12 in InspectorSlot
  VBoxContainer
    HBoxContainer header (Label name, Label coord)
    Label terrain_meta
    VBoxContainer stats (StatRow ×N)
    VBoxContainer effects (FxLine ×N, hidden if empty)
