Spec — C19 Modifier pick
Purpose
Between-wave choice modal: pick 1 of 3 modifiers (or skip). Pauses the world (modal, dim overlay). Selection feeds into RunState.modifiers array; PlayerStats recomputes.
Anchor & sizing
Centered modal, max-width 1200px. Card grid is 3 columns at desktop; collapses to 1 column under 720px. Each card min-height 320.
Inputs
picks:        Array[Modifier] (length 3)
rerolls_left: int
owned_mods:   Array[Modifier]    # for conflict detection
Outputs
signal picked(modifier_id), signal skipped, signal rerolled
Visual rules
• Effect line dots: green for adds/muls (gains), red for costs/conditions.
• Numeric format matches C14: prefix with +, ×, or units.
• Rarity color is only on the rarity label and the card's icon border — keeps overall card chrome neutral.
Edge cases
• Reroll = 0: button disabled but visible (so the affordance remains learnable).
• Conflict: red "replaces X" badge top-right; confirming shows a brief inline confirmation row before commit.
• Skip: always available; commits no modifier and may grant a small fallback (gold / heal — TBD by §3 economy).
