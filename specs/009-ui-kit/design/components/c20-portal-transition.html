C20 — Wave / portal transition

Full-screen overlay drawn between waves. Three beats: (1) intro — slabs close in, world dims, portal-pulse appears (~0.6 s); (2) hold — wave card + threat preview (~1.2 s, skippable); (3) outro — pulse expands and the next wave's enemies are spawned, slabs retract. Composes with C19 (modifier pick happens during hold if the wave grants one).

[ARENA · enemies spawning]
Spec — C20 Wave / portal transition
Purpose
Cover the spawn frame so wave changes feel narrative, not technical. Hides loading, presents threat preview, and provides a hook for C19 (modifier pick during hold).
Anchor & sizing
Full viewport. CanvasLayer above arena, below modals (C19 overlays it).
Timeline
0.0  → 0.6   intro    slabs close, dim, ring fades in
0.6  → 1.8   hold     wave card + threat preview (skippable)
1.8  → 2.0   spawn    enemies instanced behind veil
2.0  → 2.4   outro    ring expands, slabs retract, dim clears
Inputs
wave_index: int   # 1-based
wave_max:   int
wave_label: String
threats:    Array[{ icon, count }]
boss:       bool
grants_modifier_pick: bool   # if true, hand off to C19 during hold
Outputs
signal intro_done, signal hold_done, signal outro_done — TurnLoop awaits these in order.
Edge cases
• Skip during hold: jumps straight to spawn → outro. Intro is never skippable (need 0.6s for spawn-prep cover).
• Modifier pick wave: after spawn step, pause-the-world via C19; outro triggers on C19's picked / skipped.
• Boss flag: swap ring tint to --sem-damage; threat-pip "BOSS" replaces count.
Godot mapping
PortalTransition : CanvasLayer        # layer 90
  ColorRect dimmer                        # animated alpha
  ColorRect slab_top, slab_bot           # tween off-screen → 60px in
  CenterContainer
    VBoxContainer
      Label ring_label
      Control ring                       # custom-drawn pulse rings (3 concentric)
      VBoxContainer wave_card
      HBoxContainer threat_row
      Label skip_hint
