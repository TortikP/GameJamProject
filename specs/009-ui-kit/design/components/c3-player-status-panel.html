Spec — C3 Player status panel
Purpose
Always-readable mini-dossier on the player. Unlike C12 it never swaps content based on selection — it is the player's permanent ID card.
Anchor & sizing
Bottom-left of bottom bar. 320×104 fixed.
Inputs
name:      String
race:      String
hp:        int
max_hp:    int
preview:   int
speed:     int
damage_bonus: int
statuses:  Array  # same shape as C5
Edge cases
• Speed = 0 (stunned): show "0", red. Skill bar handled separately.
• Lethal preview: entire fill bar turns red; numeric stays current value with "−N" badge.
• Many statuses: mini-strip caps at 6 with "+N" overflow.
Godot mapping
