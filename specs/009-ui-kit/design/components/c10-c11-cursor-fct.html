UI KIT // C10 + C11

C10 Hex cursor · C11 Floating combat text

Two world-space primitives. C10 is the persistent hover indicator (one hex outline at the mouse). C11 is the burst-text feedback that pops out of an actor when an effect resolves — damage, heal, miss, status applied. Both are atomic enough to share a file.

C10 Hex cursor — states

Idle hover · neutral

No action queued. Plain border in --ui-text-dim; thinner than any in-action ring so it disappears under C7/C8.

Spec — C10 Hex cursor
Purpose
The single source of truth for "where is the player pointing". Other previews (C7/C8) rely on it; never let the cursor disappear during normal play.
Inputs
hovered_hex: Hex
mode:        Enum { IDLE, ACTION_VALID, ACTION_INVALID, INSPECT }
Outputs
signal hex_clicked(hex, button)
Edge cases
• Off-grid: hide cursor entirely; don't pin to nearest edge.
• Modal open: cursor is hidden by host (ModalLayer). C10 doesn't track it.
Godot mapping
HexCursor : Node2D                  # child of arena, z = 7 (above C7/C8)
  CursorOutline : Node2D            # _draw() one polygon, color/width via mode
Spec — C11 Floating combat text
Purpose
Communicate what an effect did, where, in less than a second. Tightly paired with damage/heal events from BattleResolver.
Inputs (per spawn)
actor:    Actor          # spawn anchor
kind:     Enum { DAMAGE, CRIT, HEAL, MISS, BUFF, DEBUFF }
text:     String          # "−15", "miss", "↓ bleed · 2t"
delay_ms: int = 0         # for stacking
Animation
• Lifetime: 700 ms standard, 1100 ms for CRIT.
• Motion: ease-out vertical translate −24 px; alpha 0 → 1 by 80 ms, holds, fades 0 over last 250 ms.
• Stacking: subsequent FCT on the same actor offsets X by ±10 px alternating, delayed 80 ms each.
Edge cases
• Actor dies mid-flight: FCT continues from the death spot; doesn't follow corpse fade.
• Off-screen actor: clamp spawn to arena viewport edge (don't pin to UI).
• Reduce-motion accessibility: skip rise+fade, render static text for 600 ms then dismiss.
Godot mapping
FCTLayer : Node2D                    # shared spawner; pool of N labels
  FCTLabel : Label                    # custom theme per kind, AnimationPlayer drives
