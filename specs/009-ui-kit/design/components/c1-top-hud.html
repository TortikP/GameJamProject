Spec — C1 Top HUD bar
Purpose
Persistent at-a-glance run state: whose turn, wave progress, run timer, pause access.
Anchor & sizing
Top-edge full-width, 56px high. Anchors top + left + right. Padding 24px sides.
Inputs
turn_index:   int
turn_who:     enum { player, world }
wave_current: int
wave_total:   int
run_seconds:  int
is_final_wave: bool
Outputs
signal pause_pressed()
signal help_pressed()
Edge cases
• World turn: swap turn-block to red variant; gray out Help button.
• Run timer overflow: render as HH:MM:SS past 1h.
• Wave 0 (transitioning): show — in num slot until next arena spawn.
Godot mapping
TopHud : PanelContainer                # anchors top, custom_minimum_size = (-1, 56)
  HBoxContainer                          # separation = 24
    HBoxContainer left  (size_flags = EXPAND_FILL)
      TurnBlock : PanelContainer        # StyleBoxFlat border = focus | sem-damage
        Label who, Label num
      StatBlock : VBoxContainer
        Label lbl, Label val
    WavePill : PanelContainer (center)
    HBoxContainer right (size_flags = EXPAND_FILL, halign=END)
      Button help, Button pause
