Spec — C4 HP bar (world-space)
Purpose
Persistent, always-visible HP readout above every actor. Same widget for player and enemies (Pillar 2); unifies team accent, fill color, incoming-damage preview into one cheap drawable.
Anchor & sizing
World-space, parented to the actor sprite, offset (0, -14) px from sprite top-edge. Width 30, height 4. Bar does not scale with camera zoom in jam scope (Godot: render in actor's local CanvasLayer at zoom-compensated transform).
Inputs
hp:        int     # 0..max_hp
max_hp:    int
team:      enum   { player, enemy, neutral }
preview:   int    # incoming dmg from cast preview, 0 = no preview
visible:   bool   # hidden if false (dead, fog-of-war)
Color rules
pct = hp / max_hp
if pct <= 0.15:  fill = --hp-crit
elif pct <= 0.30: fill = --hp-low
else:             fill = --hp-fill

frame outline = --team-{team}
preview width = clamp(preview / max_hp, 0, pct)  # cannot exceed current fill
Edge cases
• Lethal preview: preview width clamps to fill width — never extends into empty area.
• HP overheal > max: bar caps at 100% fill; numeric label shows true value (e.g. 120/100).
• 0/0 actor: hide bar entirely (visible=false).
• Multi-source preview: sum incoming damage from all preview sources before passing as preview input. Aggregation happens upstream.
• Tiny actor (sprite < 30px wide): bar still 30px wide, can extend past sprite silhouette. Acceptable.
Godot mapping
Custom Control with _draw() override (cheaper than 4 nested ProgressBars): 
HpBar : Control                         # custom_minimum_size = Vector2(30, 4)
  # exports: hp:int  max_hp:int  team:int  preview:int  visible:bool
  # _draw():
  #   1. fill rect (30,4) with --hp-bg
  #   2. fill rect (pct*30, 4) with chosen --hp-* color
  #   3. fill rect at right edge of fill, width=preview_pct*30, with --hp-preview
  #   4. draw_rect 1px black frame
  #   5. draw 1px team color frame outside (offset -1, +2 size)
  Label (numeric)                            # anchor bottom of HpBar, offset y=-12
                                              # theme_override_font_sizes/font_size = 10
                                              # add_theme_constant_override("outline_size", 2)
Connect to actor signals: hp_changed(new), preview_changed(amount). queue_redraw() on either.
